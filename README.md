# ⚡ Installation

## Clone Repository

```bash
git clone <your-repo-url>
cd institutional-fundamental-analysis-india
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run Pipeline

```bash
python main.py
```

This will:

- Load stock universe
- Fetch yfinance data
- Generate institutional scores
- Train ML model
- Run backtests
- Create DuckDB database
- Export Excel analytics

---

# 🌐 Run Dashboard

```bash
streamlit run streamlit_app.py
```
